# CSharp-Day-2

Welcome to **CSharp-Day-2**, a hands-on learning repository designed to reinforce fundamental C# programming concepts through practical exercises. This repository serves as part of a structured learning journey, where each day focuses on a specific topic to build a strong foundation in C# and .NET development.

 Structure

The repository is organized into folders and files that correspond to key C# topics covered on Day 2:

- **`BoxingUnboxing`** — Demonstrates the concepts of boxing and unboxing in C#.
- **`ClassWithFMC`** — Explores class definitions, fields, methods, and constructors.
- **`DataTypes`** — Covers built-in and custom data types in C#.
- **`C# day 2.sln`** — The Visual Studio solution file to compile and run all exercises together.

 Learning Objectives

By working through this repository, you will:

- Understand how value types and reference types interact via boxing and unboxing.
- Learn how to define and use classes with fields, methods, and constructors.
- Gain familiarity with C# data types and their appropriate use cases.
- Practice writing and running C# code in a structured project environment.

 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DibyaGit/CSharp-Day-2.git
   ```

2. Open the solution file `C# day 2.sln` in Visual Studio or Visual Studio Code with the .NET SDK installed.

3. Build and run each project to explore the concepts in action.
